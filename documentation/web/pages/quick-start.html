Quick start

A minimal static truss deck that exercises every core concept in one file.

Copy the deck below, save it as truss.fem, and run FEMaster against it. If it completes without error you have a working installation.

** Two-bar truss — quick start deck
*NODE
1, 0.0, 0.0, 0.0
2, 1.0, 0.0, 0.0
3, 0.5, 0.5, 0.0

*ELEMENT, TYPE=T3D2
1, 1, 3
2, 2, 3

*NSET, NAME=PIN
1, 2
*NSET, NAME=LOAD_PT
3

*MATERIAL, NAME=Steel
*ELASTIC
210000.0, 0.3

*TRUSSSECTION, ELSET=ALL, MATERIAL=Steel
0.01

*SUPPORT, NSET=PIN, DOF=1,2,3
*CLOAD, NSET=LOAD_PT
0.0, -1000.0, 0.0

*LOADCASE, TYPE=LINEARSTATIC, NAME=LC1
  *LOADS
    LOAD_PT
  *SUPPORTS
    PIN
  *SOLVER, TYPE=DIRECT

What this deck covers

*NODE / *ELEMENT

Coordinate rows and connectivity. TYPE=T3D2 is a two-node truss in 3-D.

*NSET

Named node groups keep load and support definitions independent of raw IDs.

*MATERIAL / *ELASTIC

Material context with isotropic elastic data: Young's modulus and Poisson's ratio.

*TRUSSSECTION

Assigns cross-sectional area and material to the truss element set.

*SUPPORT / *CLOAD

Collector definitions. Not active until referenced inside a *LOADCASE.

*LOADCASE

Activates collected boundary data and runs a linear static solve with a direct solver.

Running FEMaster

femaster truss.fem

Output is written to truss.res. See the result format page for how to read it.
