Installation

Build prerequisites, CMake configuration, and a smoke test to verify your FEMaster binary.

Prerequisites

C++ toolchain

A compiler with C++ support matching the project's CMake configuration. On Windows, Visual Studio generator workflow is used in the existing build tree.

CMake

Configure the project from the repository root and build the solver target from the generated build directory.

Build

Run these two commands from the repository root:

cmake -S . -B build
cmake --build build --config Release

If you already use the repository's existing build directory, keep using that generator and configuration — do not mix build systems in the same directory.

Smoke test

After building, run a small input deck to verify the binary works. The quick-start page has a minimal static truss deck that exercises nodes, elements, material, section, supports, loads, and a loadcase.
