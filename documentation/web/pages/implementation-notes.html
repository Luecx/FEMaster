Implementation notes

Solver behaviours that are easy to miss from syntax alone.

Active DOFs

The active index matrix is built from model objects — not from nodes alone. A node that is not referenced by any element, load, support, constraint, or feature will not create any active degrees of freedom.

Constraint reports

*CONSTRAINTSUMMARY prints the generated constraint equations to stdout. It is a diagnostic-only command — it does not modify the constraint set or affect the solve.

Nonlinear positions

*NONLINEARSTATIC updates current node positions during assembly at each iteration but restores the original model positions after the run completes. Subsequent loadcases see the original geometry.

Contact tangent

The contact tangent stiffness is penalty-based. Closest-point coordinates and the contact normal are fixed during a single assembly pass. They are updated between Newton iterations.
