Recommended reading order

1Install or build FEMaster and run the quick-start deck to verify the binary.
2Read the DSL overview to understand commands, scopes, keywords, and data lines.
3Work through groups in order: model → properties → loads/supports → constraints → loadcases.
4Use the analysis controls reference when setting up solver parameters and output requests.
5Check the result format page before post-processing .res files.
